# Realtime Data Analysis with Kafka streaming, Python and AWS

🔺This project demonstrates a real-time data streaming pipeline for processing and analyzing event data using Apache Kafka, Python, and AWS services. It simulates high-volume transactional data (e.g., e-commerce events) and enables near real-time analytics and anomaly detection.🔺

⚙️ Tech Stack
Python – Data generation and processing\n
Apache Kafka – Real-time data ingestion \n
AWS Glue – Stream processing \n
Amazon S3 – Data lake storage \n
Amazon Athena – Querying and analytics
